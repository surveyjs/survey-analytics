Lazy Loading Choices Example

Scenario: This example demonstrates how visualizers handle questions with lazy loading enabled.

The country dropdown has 1000+ choices but only the first 25 are loaded initially via lazy loading. However, the survey response data contains values (Albania, France, Zimbabwe) that span the full range of choices, including some outside the initially loaded 25.

Solution: The visualizer now shows ONLY the values that actually appear in the survey response data, rather than showing all loaded choices. This keeps the visualization focused on actual responses and works efficiently with lazy loading. Values are displayed using their choice labels when available, or using the value itself as the label if no choice definition exists.
